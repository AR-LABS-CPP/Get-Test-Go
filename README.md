## Description
Get Test Go is a testing and recruitment system that showcases the use of digital assessments along with IQ and EQ tests to assess and recruit candidates.

## Technologies Used
- React.js
- Express.js
- PostgreSQL
- Json Web Tokens
- Tailwind CSS
